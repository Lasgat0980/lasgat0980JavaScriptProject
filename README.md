# Developer Portfolio - Lastazia Dickens

This is a professional portfolio website showcasing my journey as an aspiring Software Developer at ECPI University.

## Languages & Technologies Used
* **HTML5**: Structure and Semantic Layout
* **CSS3**: Custom styling, Dark Mode toggling, and Tooltip animations
* **JavaScript (ES6)**: Dynamic DOM manipulation, Session Storage, and JSON handling

## Key Features
* **Dynamic Project Rendering**: Projects are stored as JSON in `sessionStorage` and rendered via JS loops.
* **Persistent Dark Mode**: Saves user theme preference using `localStorage`.
* **Interactive UI**: Includes a custom welcome modal and automated notification system.

## How to Run
1. Clone the repository.
2. Open `index.html` in any modern web browser.
3. No external dependencies or libraries are required (Vanilla JS).
